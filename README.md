-----Skidding in the 90s-----

A simple racing game with a win/lose cycle.

Controls

Upper arrow	-> Accelerate
Bottom arrow	-> Brake / Move backwards
Left arrow 	-> Turn left
Right arrow 	-> Turn right
Spacebar 	-> Restart lap

Debug Features

F1 	-> Allow camera rotation
F2 	-> Reset game (Last lap time and best lap time remain)

Mechanics:

Static obstacles	-> If you crash with one when your speed is higher than 70 km/h, you restart the lap.

Dynamic obstacles	-> They can slow you down or even kick you out of the road.

Moving backwards	-> You can move backwards or slow down with the bottom arrow key.

Void			-> If you fall off the road, you restart the lap.

Win / lose condition	-> If you successfully complete 3 laps before 5 minutes, you win. If not, you lose.
